# <p align="center">Python</p>
- ### For Loop
```python
for i in [1,2,3]:
	print(i)

for i in "Hello":
	print(i)

for i in range(10):
	print(i)

```


- ### While Loop
```python
i = 0;
while i < 10:
	print(i)
	i += 1
```

<br>


- ### Infinite Loop
```python
i = 0;
while True:
	print(i)
	i += 1
```

<br>

- ### Loop With Else
```python
i = 0
while i > 10:
	print(i)
	i += 1
else:
	print(i)
```


- ### For In Loop
```python
person = {
	"fname":"Jagadish",
	"lname":"Chakma",
	"age": 28
}

for key in person:
	print(person[key])
```

- ### For Of Loop
```python
let nums = [1,2,3,4,6]
for num in nums:
	print(num)
```

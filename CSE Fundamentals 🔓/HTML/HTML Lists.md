- HTML lists allow web developers to groupt a set of related items in lists.
- ### Unordered Lists
```html
<ul>
	<li>list one</li>
	<li>list tow</li>
</ul>
```
- ### Ordered Lists
```html
<ol>
	<li>list one</li>
	<li>list two</li>
</ol>
```
- ### Description Lists
```html
<dl>
	<dt>html</dt>
	<dd>Hyper text markup language</dd>
	<dt>javascript</dt>
	<dd>A web standard language.</dd>
</dl>
```

- An HTML forms is used to collect user input. The user input is most often sent to a server for processing.
```html
<form action="" method="" enctype="" target="" autocomplete="">
	<input type="text" id="" name="" value="" placeholder=""/> <!-- Uses for text input -->
	<input type="radio" id="" name="" value=""/> <!-- Uses for choose -->
	<input type="checkbox" id="" name="" value=""/> <!-- Uses for checked -->
	<input type="submit" id="" name="" value=""/> <!-- Uses for submit form -->
	<input type="button" id="" name="" value=""/> <!-- Uses for make a button -->
	<select id="" name="">
		<option value=""></option>
	</select> <!-- Select your desired options -->
	<textarea id="" name="" rows="" cols="">Your text here...</textarea>
</form>
```

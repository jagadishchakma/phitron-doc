- # The Math Module
- Python has also a built in module called ==math==, which extends the list of mathematical functions.
- To use it, you must import the ==math== module.
```python
import math
```
- # Square Root
```python
import math
math.sqrt(64)
```
- # Number Rounds
```python
import math
math.ceil(2.4)
math.floor(3.4)
```
- # PI
```python
import math
x = math.pi
print(x)
```
- # Power
```python
import math
power = math.pow(5,6)
print(power)
```

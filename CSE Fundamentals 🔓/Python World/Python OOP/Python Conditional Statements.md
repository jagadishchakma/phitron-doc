- # If...elif...else
- Avoid brackets.
- Manitain Indentation.
```python
if 2 == 1:
	print("something")
elif 2 > 3:
	print("something")
else:
	print("default")
```
- # Short Hand If
- Also called this  is ternary opertor.
```python
if 2 == 2: print("Yes they are equal")
```
- # Short Hand If...Else
- Also called this is ternary opertor.
```python
print("Yes, they are equal") if 2 == 1 else print("No, they are not equal")
```

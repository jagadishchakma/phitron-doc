- ### Union
```mysql
select * from student where age > 18
union
selection * from employee where salary > 10000
```
- ### Union All
```mysql
select * from student where age > 18
union all
selection * from employee where salary > 10000
```
- ### Intersect
```mysql
select * from student where age > 18
intersect
selection * from employee where salary > 10000
```
